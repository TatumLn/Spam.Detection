# SpamGuard - Détecteur de Spam Intelligent

Application web de détection de spam utilisant le Machine Learning (Naive Bayes) avec une précision de 96%. Analysez vos messages en temps réel et identifiez instantanément les spams.

## Demo

**[Tester l'application](https://spamadetection.netlify.app/)**

## Technologies

| Frontend | Backend | ML & Data | DevOps |
|----------|---------|-----------|--------|
| React 19 | Flask 3 | Scikit-learn | Netlify |
| Vite | PostgreSQL | Pandas | Render |
| Tailwind CSS | JWT Auth | NLTK | GitHub |
